** aws-auth
terraform -chdir=provider/aws/tier/env/aws-auth init
terraform -chdir=provider/aws/tier/env/aws-auth import kubernetes_config_map.aws-auth kube-system/aws-auth
terraform -chdir=provider/aws/tier/env/aws-auth state rm module.aws-auth-config-map.data.kubernetes_config_map.aws-auth
terraform -chdir=provider/aws/tier/env/aws-auth state list
terraform -chdir=provider/aws/tier/env/aws-auth apply -auto-approve
note: never ever try to destroy this (it will destroy the aws-auth, you'll never go back!)
note: removing resource from tf state,
List all state: terraform state list
Remove desired resource from state: terraform state rm <name>
verify via: kubectl edit -n kube-system configmap/aws-auth
